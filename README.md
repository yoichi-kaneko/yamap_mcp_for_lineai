# yamap_mcp_for_lineai

YAMAP の登山計画ページからデータを取得するための MCP (Model Context Protocol) サーバーです。

## 概要

自作の LINE AI「碧衣（あおい）」から YAMAP の山行計画情報を取得・活用するために作成しました。
完全に個人用途を前提としており、汎用性は一切考慮していません。

内部では Playwright (Chromium) を使って YAMAP の登山計画ページにアクセスし、レンダリング済みの HTML をスクレイピングすることでデータを取得しています。
そのため、**YAMAP 側のページ構造やアップデートによって、予告なく動作しなくなる可能性が常にあります。**

## 事前準備

本 MCP サーバーを実行する環境にて、事前に以下のコマンドで Playwright の Chromium をインストールしてください。

```bash
npx playwright install chromium
```

## 実装済みツール

| ツール名 | 概要 |
|---|---|
| `fetch_yamap_plan_page` | YAMAP の山行計画ページの URL を受け取り、概要・計画データ・移動計画などの情報を取得する |
